# Prompt para crear una web de empresas representadas

```text
Crea una pagina web profesional en espanol para mostrar las empresas que represento.

La web debe tener un diseno moderno, claro y elegante. Quiero que transmita confianza, experiencia y profesionalidad.

Estructura de la pagina:
1. Encabezado con el titulo: "Empresas que represento"
2. Seccion principal con una frase de presentacion, por ejemplo:
   "Trabajo con empresas de confianza para ofrecer soluciones de calidad a mis clientes."
3. Una seccion con tarjetas para varias empresas representadas. Cada tarjeta debe incluir:
   - Nombre de la empresa
   - Logotipo o imagen representativa
   - Breve descripcion
   - Sector o especialidad
   - Boton de contacto o mas informacion
4. Seccion "Sobre mi", explicando que actuo como representante comercial o colaborador.
5. Seccion de contacto con telefono, correo electronico y enlace a WhatsApp.
6. Pie de pagina sencillo.

Estilo visual:
- Fondo claro
- Colores profesionales
- Tipografia legible
- Tarjetas bien ordenadas
- Diseno adaptable a movil
- Aspecto serio, limpio y comercial

Crea el codigo completo en un solo archivo index.html, incluyendo HTML y CSS.
Usa empresas de ejemplo que luego pueda cambiar por las reales.
```
